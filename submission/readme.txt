- Version Control:
    Please install the correct version of the packages mentioned in 'requirement.txt'

- Usage:
    python main.py --data_dir="path\to\data\directory"

- Process flow:
    Running main.py script will train the model using the train dataset and produce labels using the test dataset.
    The result file will be available at 'model_output' folder under the provided directory in --data_dir argument
